1) Run only using Dev C++ (if you plan to edit source code)
2) Open Project1.dev to open source code
3) All necessary prequesites are included in the root folder
4) Always run as a 32-bit package compilation
5) Add the following Linker file in project parameters as well as in compiler options to make the game working

-lbgi
-lgdi32
-lcomdlg32
-luuid
-loleaut32
-lole32

NOTE:-

Run Project1.exe to play the game without the need of the compiler
